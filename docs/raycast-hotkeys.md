# Raycast Hotkey Configuration

Global keyboard shortcuts for quick app access via Raycast.

## Setup

1. Open Raycast → Settings (⌘,)
2. Go to Extensions → Applications
3. Add hotkeys for each app below

## Hotkeys

| Shortcut | App | Note |
|----------|-----|------|
| `⌥ Space` | Raycast | Main launcher (replaces Spotlight) |
| `⌃⌥B` | Arc | Browser |
| `⌃⌥P` | PHPStorm | IDE |
| `⌃⌥O` | Obsidian | Notes |
| `⌃⌥F` | Finder | File manager |
| `⌃⌥G` | Ghostty | Terminal |
| `⌘\`` | Ghostty | Quick dropdown (configured in Ghostty settings) |

## Pattern

All app hotkeys use `Ctrl + Option + Letter`:
- **B**rowser → Arc
- **P**HPStorm → IDE
- **O**bsidian → Notes
- **F**inder → Files
- **G**hostty → Terminal

## Additional Raycast Extensions

Consider installing these extensions:
- **Clipboard History** - Access clipboard with `⌃⌥V`
- **Window Management** - Quick window tiling
- **GitHub** - Quick access to repos and PRs
- **1Password** - Search passwords

## Disabling Spotlight

After setting up Raycast:
1. System Settings → Keyboard → Keyboard Shortcuts
2. Spotlight → Uncheck "Show Spotlight search"
3. This frees up `⌘Space` if you prefer it for Raycast
